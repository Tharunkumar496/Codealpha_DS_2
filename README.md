## 📌 Project Overview
This project demonstrates the fundamentals of Data Analysis using Python. It involves loading a dataset, exploring its structure, checking for missing values, and generating descriptive statistics to understand the data.

- Import and load dataset using Pandas
- Display dataset information
- Check data types
- Handle missing values
- Generate summary statistics
- Explore dataset structure
- Basic Exploratory Data Analysis (EDA)
